# vantapad

![vantapad](https://imgur.com/a/vantapad-isometric-angle-QZpQ3jn)

A Hackpad made as a part of Hackclub's Blueprint program. VantaPad is a sleek black macropad I will use for controlling simple functions on my computer that I currently do by hand. It is one of my first hardware projects and it taught me many valuable skills such as schematic design, PCB design, Firmware programming, and more.
